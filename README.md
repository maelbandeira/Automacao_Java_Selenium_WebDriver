# Testes com Selenium WebDriver Java

Projeto de estudos em testes automatizados 

### Frameworks/ Versões :
    
    1. Maven = Gerenciamento de dependências e build de aplicações Java.
    2. JUnit 4.12 = Encarregado pela estruturação dos casos de teste.
    3. Selenium WebDriver 4.1.0 = Ferramenta que ficará responsável por automatizar as entradas de dados no navegador.
    4. Cucumber = Ferramenta de teste que suporta BDD(Behavior Driven Development),desenvolvimento orientado por comportamento.
